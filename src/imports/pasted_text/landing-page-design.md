Saya sedang membangun Conversion Funnel untuk klinik kesehatan / estetika yang terdiri dari 3 halaman utama:
1.	Landing Page
2.	Booking Page
3.	Confirmation Page
Namun pada tahap ini saya hanya ingin membangun Landing Page terlebih dahulu, karena frontend development dilakukan per halaman secara terpisah. 
Bangun Landing Page funnel yang profesional, premium, dan high-conversion. Landing page ini bukan website profil, tetapi conversion interface yang bertujuan:
Mengubah pengunjung menjadi booking konsultasi / treatment.
CTA utama harus mengarahkan pengguna ke Booking Page.
