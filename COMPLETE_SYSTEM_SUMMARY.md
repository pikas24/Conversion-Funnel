## 🔗 Backend Integration Details

### API Endpoint: POST /api/bookings

**Request Body:**
```json
{
  "full_name": "John Doe",
  "phone_number": "+628123456789",
  "email": "john@example.com",
  "service_interest": "Laser Treatment",
  "preferred_date": "2026-03-17",
  "preferred_time": "14:00",
  "notes": "Skin concern details..."
}
```

**Response:**
```json
{
  "status": "success",
  "message": "Booking created successfully",
  "booking_id": "BOOK-1234567890",
  "lead_id": "LEAD-9876543210",
  "appointment_id": "APT-5555555555"
}
```

**Validations Applied:**
- Full name: Min 3 chars
- Phone: +62 format, 9-13 digits
- Email: Valid format (optional)
- Service: Required, from predefined list
- Date: Today or future
- Time: Valid time slot

**Backend Actions:**
1. Validates all fields
2. Calculates lead score
3. Creates Lead record in Airtable
4. Creates Appointment record in Airtable
5. Sends Telegram notification to admin
6. Sends Email confirmation to patient
7. Returns booking_id

**Error Handling:**
- Network errors caught
- Validation errors displayed
- User-friendly messages
- Console logging for debugging
